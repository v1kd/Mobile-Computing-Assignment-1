Usage
------
-> Open the application
-> Click on "Run" to start the graph.
   The graph renders and then it runs
-> To pause the graph click on "Pause" button
-> To resume the graph click on "Run" button
-> To stop the graph any time click on "Stop" button
-> Select the text field to fill form data
